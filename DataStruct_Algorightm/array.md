# 数组

数组是一种线型表数据结构，它用一组连续的内存空间来存储一组具有相同类型的数据

线型表：数据排成一条线一样的结构，每个线型表上的数据最多只有前和后两个方向

数组、链表、队列、栈等都是线型表结构
